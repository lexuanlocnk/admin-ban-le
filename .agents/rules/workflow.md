# Workflow Rules

- Inspect the target screen and one similar screen in `src/views` before editing.
- If a page is user-facing in the admin, also check `src/routes.js` and `src/_nav.js`.
- Keep changes local to the target module unless the same pattern is already duplicated elsewhere.
- Preserve existing query-param, pagination, toast, and permission-check behavior.
- Update `session/latest_summary.md` with a short factual handoff when the task ends.
